FR24 Feeder Dashboard

A supercharged, standalone UI for your local Flightradar24 Raspberry Pi feeder. Features include live Planespotters.net image integration, distance sorting, and dynamic telemetry.

⬇️ Install Script
Requirements:
  • Browser extension like Tampermonkey installed.
  • Your feeder must be located at 192.168.50.252:8754 (Edit the script if your IP changes).
